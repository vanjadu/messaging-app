# React, Express & Socket.io Messaging App

Full Stack App made with socket.io that lets you chat with someone else.

## Tech stack

Front end:

`React`

`socket.io-client`

`react-icons`

`TailwindCSS`

`DaisyUI`

Back end:

`Express`

`socket.io`

`Cors`

`Colors`

`nodemon`
